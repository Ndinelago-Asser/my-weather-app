# Weather App 

A simple weather app built with **HTML, CSS, and JavaScript** as part of the SheCodes course.  
Search for any city to get the current weather, temperature, and weather description.

## Live Demo
Check it out here: [Weather App on Netlify](https://amazing-taffy-08457f.netlify.app)

## Features
- Search for any city worldwide
- Displays current temperature and weather description
- Responsive design for desktop and mobile
- Dynamic updates using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
